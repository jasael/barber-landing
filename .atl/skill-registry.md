# Skill Registry

This project uses Spec-Driven Development (SDD) to implement high-quality features.

## Project Standards

### Tech Stack
- **Framework**: Astro (v6)
- **Styling**: Tailwind CSS (v4) with CSS Animations & Shapes
- **Components**: Astro Components
- **Language**: TypeScript

### Conventions
- **Component Architecture**: Atomic design, keeping layouts, pages, and components distinct.
- **Design Language**: Minimalist, innovative, premium design with dark and woody (amaderados) tones. Using rich dark browns and blacks as base colors with warm gold/copper accents.
- **Responsive Layout**: Mobile-first, completely fluid responsive design.
- **Interactive Elements**: Unique ID attributes on all buttons, forms, and interactive links to facilitate automated browser testing.
- **SEO Best Practices**: Proper title tags, meta descriptions, single unique H1 tag, semantic HTML (header, nav, main, section, footer), sitemap, and schema metadata.
